## Implementation

This is an independent implementation of the CCRT evaluation framework based on the published paper.

## Features

- **Genetic Algorithm Search**: Automated discovery of effective prompts for concept erasure evaluation
- **GPT-4V Scoring**: Vision-language model-based assessment of concept erasure quality
- **Vocabulary Management**: Structured concept vocabulary for testing
- **Residual Testing**: Evaluation of remaining traces of erased concepts
